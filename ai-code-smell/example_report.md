## P1 — Cost trap / silent bug

### P1.1 [L1] LLM call without max_tokens — runaway cost risk
- **Location:** `src/lib/claude.ts:42`
- **Evidence:** `client.messages.create({ model, messages })`
- **Fix:** Add `max_tokens=1024` (or appropriate cap). Cursor users have reported $1,400+ bills from runaway output. Never call an LLM without an output bound.

### P1.2 [A1] await without enclosing try/catch (unhandled promise rejection)
- **Location:** `src/pages/api/checkout.ts:18`
- **Evidence:** `const session = await stripe.checkout.sessions.create({ ... })`
- **Fix:** Wrap the await in try/catch, or chain `.catch()` on the promise. Unhandled rejections crash Node 15+.

### P1.3 [A2] `except: pass` silently swallows exceptions

- **Evidence:** `except Exception:\n    pass`
- **Fix:** At minimum log the exception with stacktrace. Preferably handle the specific error.
