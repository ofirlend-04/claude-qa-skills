# ai-code-smell — Claude QA Skill

Detects patterns specific to AI-generated code (Claude / Cursor / Copilot).

**Flagship detection:** slopsquatting — hallucinated npm/pip packages that attackers
register with malicious payloads. LLMs hallucinate package names 5–33% of the time
(GitGuardian 2026). Live attacks in the wild.

## What it catches

| Rule | Severity | What |
|------|----------|------|
| S1   | P0       | Hallucinated npm / pip package (registry 404) |
| S2   | P1       | Suspicious new package (<30 days, <100 dl/wk on npm) |
| A1   | P1       | `await` without try/catch |
| A2   | P1       | Empty catch / `except: pass` |
| L1   | P1       | LLM call without `max_tokens` |
| L2   | P1       | Hallucinated API method (fetch.post, dict.has_key, ...) |
| R1   | P2       | `React.FC<>` deprecated |
| R2   | P2       | Class component in new code |
| R3   | P1       | `componentWillMount` / `componentWillReceiveProps` |
| D1   | P1       | `console.log` in production src |
| D2   | P2       | `TODO` / `FIXME` / `HACK` / `XXX` |
| U1   | P2       | Unused named import |
| E1   | P1       | Env var referenced but not declared in `.env*` |
| C1   | P2       | Duplicate 10-line block in 3+ places |
| M1   | INFO     | AI-assistant banner (`// Generated by Cursor`) |

## Run

```bash
python3 auto_audit.py /path/to/project
python3 auto_audit.py /path/to/project --offline
python3 auto_audit.py /path/to/project --no-duplicates
```

## Output

- `report.md` — human-readable
- `report.json` — machine-readable
- stdout — `[PX] path:line — msg` lines (parsed by `scan_all.py`)

## Exit codes

- `0` — scan succeeded, 0 findings or INFO-only
- `1` — ≥1 P1 finding
- `2` — ≥1 P0 finding (gate CI here)

## Offline mode

`--offline` skips the npm/PyPI registry calls. Use in air-gapped CI or when
rate-limited. Registry responses are cached in `.cache/npm-registry.json` /
`.cache/pypi-registry.json` for 7 days, so a fresh online run warms the cache
for subsequent offline runs.

## False-positive control

- `.qaignore` at repo root (one glob per line).
- Inline: `// qa-ignore: L1,A1` — applies to that line and the next.
